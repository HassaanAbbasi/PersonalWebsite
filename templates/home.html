Projects

Diqtionary

"Diqtionary" is a Python program that returns the definitions of words. The first version I released used a command line, while the second version used a GUI made with tkinter. Diqtionary also has a spell check feature, which tries to correct misspelled words (ex. rainnn to rain). The inspiration behind this project was to give my little brother a tool to help him with his weekly spelling and definitions homework.

Scotiabank Stocks Graph

To explore the relationship between technology and finance, I developed "Scotiabank Stocks Graph". By using Pandas to collect the stock market data for Scotiabank Toronto, and Bokeh to create a candlestick chart for said data, I enhanced my knowledge on how Python can be used to study the stock market. The graph I created shows daily stock data from January 1995 to the present. I chose to specifically plot Scotiabank Toronto's data because during my stock market research I found that it is an excellent long-term core holding. Also, Scotiabank is Canada's most international bank, which makes it all the more interesting to follow and study.

World PokeStops Map

In order to store interesting PokeStop locations on a map, I created "World PokeStops Map". Its a Python program that plots the aforementioned locations on a map generated with Folium. The locations are stored in a CSV file, which are read by a Python file to make an HTML document that contains the map. I wish to travel around the world more in the near future, so I made this project in hopes of finding cool places to visit while I play Pokemon GO abroad.
