Projects

Diqtionary

"Diqtionary" is a Python program that returns the definitions of words. The first version I released used a command line, while the second version used a GUI made with tkinter. Diqtionary also has a spell check feature, which tries to correct mispelled words (ex. rainnn to rain). The inspiration behind this project was to give my little brother a tool to help him with his weekly spelling and definitions homework.
